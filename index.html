Usage

1. Include on the page
<link href="PATH_TO/table-gistogram.css" type="text/css" rel="stylesheet">
<script src="PATH_TO/table-gistogram-directive.js"></script>
2. Add as a dependancy in Angular application
angular.module('testApp', ['tableGistogram']);
